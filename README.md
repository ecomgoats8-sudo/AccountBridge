# Agency v2

A modern agency website built with React, TypeScript, and Vite.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- ESLint

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Features

- Modern, responsive design
- TypeScript support
- Fast development with Vite
- Styled with Tailwind CSS
